## What is a Nuke bot?

A Discord nuke bot is essentially an automated script designed to rapidly and massively disrupt or destroy a server. It typically uses a bot token with administrative permissions to delete channels and roles, create large numbers of new channels, spam messages or mentions, and sometimes ban or kick members in seconds. Because it operates through Discord’s API, it can perform actions much faster than a human user could manually. In practice, it’s less of a “hack” and more of an abuse of granted permissions—if a bot (or compromised account) has high-level access, it can systematically dismantle a server almost instantly.

## What is a Webhook Abuse bot?

A Discord webhook abuse bot is a tool designed to exploit Discord webhooks in order to spam, flood, or disrupt a server. Webhooks are normally used for legitimate purposes, like sending automated updates or announcements into a channel through a special URL, but if that URL gets into the wrong hands, it can be misused. An abuse bot will repeatedly send large amounts of messages, embeds, or pings through the webhook, sometimes cycling requests to push past rate limits, which can overwhelm a channel and cause chaos. Since webhooks don’t require a full bot account to send messages, they can be easier to abuse if they aren’t properly secured or deleted after use.
